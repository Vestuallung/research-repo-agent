# Project Outline Template

Use this template for a small research reading or reproduction project.

## 1. Background

Explain the topic in plain language. State why the problem matters and what larger area it belongs to.

## 2. Problem Definition

State the exact research question. Define what is in scope and what is out of scope.

## 3. Related Work Map

Group papers by role:

- foundational problem papers;
- method papers;
- benchmark or dataset papers;
- system or efficiency papers;
- critique, failure, or limitation papers.

For each group, write the shared idea and the main difference between papers.

## 4. Method Comparison

Compare methods by input, output, assumptions, cost, evidence type, and limitations.

Avoid ranking methods unless the same benchmark and evaluation setting are available.

## 5. Experiment Or Demo Design

Describe the smallest runnable experiment.

Include data source, command, output file, metric, and expected failure modes.

## 6. Results Record

Record what actually happened.

Separate raw output, metric summary, and interpretation.

## 7. Agent Drafting Log

List what the Agent generated and which source files it used.

Mark each generated section as `draft`, `needs_review`, `checked`, or `final`.

## 8. Limitations

State missing papers, weak evidence, untested assumptions, and experiment limits.

## 9. Final Takeaways

Write only conclusions supported by reviewed notes or experiment outputs.

Keep future ideas separate from established findings.
